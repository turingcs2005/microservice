Geography APIs

All APIs below use GET

  • City validator: check if a city belongs to a state
  • State validator: check if a sate abbreviation is a valid US state
  • Postal code validator: check if a ZIP code is a valid US ZIP code
  • City list: return an array of city names for a given state
  • State list: return an array of US state name and state abbreviation
  • ZIP code list: return an array of US ZIP codes given a state abbreviation and city name

URL and parameters

  • '/api/geography/validatePostCode/:postCode'
  • '/api/geography/cityArray/:state'
  • '/api/geography/stateArray'
  • '/api/geography/validateState/:state'
  • '/api/geography/validateCity/:state/:city'
  • '/api/geography/zipCodeArray/:state/:city'

If your city name has special character(s), you need to encode them (either via express's urlencoded middleware, or manually) correctly following URL encoding standard.

For example, if you want to check if 'Fall River' is a valid city in Massachusetts using your browser, use '/api/geography/validateCity/MA/Fall%20River'.
